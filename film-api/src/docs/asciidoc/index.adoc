== Posts

=== 게시물 생성

`*Request*`
----
POST /api/v1/posts HTTP/1.1
Content-Type: multipart/form-data; boundary=6o2knFse3p53ty9dmcQvWAIx1zInP11uCfbm
Authorization: Bearer ${token_value}
Host: localhost:8080

Content-Disposition: form-data; name=files
files = file []

Content-Disposition: form-data; name=com
{
 "title": "TEST 제목",
 "previewText": "미리보기 TEST",
 "latitude": "31.12312",
 "longitude": "144.4444",
 "availableAt": "2021-12-31",
 "content": "내용 TEST"
}
----
|===
|Path|Type|Description

|`+files+`
|`+Array+`
|게시물 사진 리스트

|`+files.[]+`
|`+File+`
|게시물 사진

|`+com+`
|`+String+`
|생성요청 String(json형식)

|`+com.title+`
|`+String+`
|게시물 제목

|`+com.previewText+`
|`+String+`
|게시물 엿보기 문구

|`+com.latitude+`
|`+String+`
|위도

|`+com.longitude+`
|`+String+`
|경도

|`+com.availableAt+`
|`+String+`
|열람가능 날짜

|`+com.content+`
|`+String+`
|게시물 내용
|===
